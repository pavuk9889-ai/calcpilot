# Security Policy

## Supported versions

CalcPilot is currently in MVP stage. The latest public version of the website is supported.

Current public version:

- CalcPilot v1.3

## Reporting a vulnerability

If you find a security issue, please contact:

geometry9889@gmail.com

Please include:

- a short description of the issue;
- affected URL;
- steps to reproduce the issue;
- browser/device information if relevant.

Please do not publicly disclose the issue until it has been reviewed.

## Scope

The current version of CalcPilot is a static website hosted on GitHub Pages.

The project currently does not include:

- user accounts;
- passwords;
- payment processing;
- personal dashboards;
- backend API;
- database storage;
- forms for collecting personal data.

## Notes

CalcPilot is an informational calculator service. All calculations are approximate and are not financial, legal, tax or investment advice.
